VPG_OCR - LEIA-ME

1) COMO INSTALAR NO COMPUTADOR

- Abra 1_instalacao e clique em 🛠️_CLIQUE_AQUI_PARA_INSTALAR.bat.
- Se o Windows mostrar a janela de segurança do arquivo .bat, clique em EXECUTAR.
  Se desejar, marque a opção para não mostrar novamente.
- Se o Python não estiver instalado, o instalador tentará instalar automaticamente o Python 3.14 usando winget.
- Quando a janela do instalador do Tesseract abrir, selecione o idioma do instalador como: English.
- Depois siga normalmente os passos até concluir.
- O instalador também tentará ajustar automaticamente o PATH do Windows para o Tesseract.

2) COMO USAR O OCR

- Abra 2_utilizacao.
- Coloque os PDFs em ENTRADA_coloque_aqui_seus_pdfs.
- Clique em 🚀_CLIQUE_AQUI_PARA_USAR_OCR.bat.
- Os arquivos processados irão para SAIDA_pdfs_processados.
